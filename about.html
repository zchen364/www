About CRUDList
A "crud" Web Application

CRUDList is a a project created by Alex Mai, Adam Livingston, and Zhiming Chen for Computer Science 210 (Web Programming) with Professor Philip Guo. It is designed to be a dynamic, responsive "CRUD" application that allows users to create accounts and find services around the U of R campus.
